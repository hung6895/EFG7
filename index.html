Instructions

This web application visualizes the local structure of self-similar fractals using virtual magnification.

Select a fractal from the dropdown and click "Start Visualization" to begin. The visualization will show magnifications of the fractal structure.

Note: Fractal data files (.mat) need to be converted to JSON format. Use the converter tool or load JSON files directly.
